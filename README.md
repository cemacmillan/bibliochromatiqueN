# Open License Library

**An archive of creator-friendly, safety-minded licenses**

This repository collects open licenses that help artists, modders, researchers, and
educators share their work freely **without surrendering attribution or safety**.

Each license here can be used exactly as you would use source code:
copy it, adapt it, fork it, or import it into your own repository, subject to its terms.

## âš™ï¸ Using a license

1. Copy the license text into your project's root as `LICENSE` or `LICENSE.md`.  
2. Keep the **SPDX header line** at the top (note: this SPDX identifier is our own; please change the "CAL" portion to be unique for your license):

   ```
   SPDX-License-Identifier: LicenseRef-CAL-1.0
   ```

3. Mention the identifier in your `README`, manifest, or mod metadata.  
4. If you wish to include the mutual non-endorsement clause, use the `-N` variant:

   ```
   SPDX-License-Identifier: LicenseRef-CAL-1.0-N
   ```
